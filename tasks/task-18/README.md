# Prometheus

Добавьте к серверу `server.js` сборку метрик для `prometheus`.

Он должен собирать дефолтные метрики ноды, а так же Histogram для API запросов.

Сервер метрик должен отвечать на http://localhost:8000/json и отдавать метрики в JSON.

Для формирования метрик используйте библиотеку `prom-client`.

У Histogram должно быть название `api_request` и бакеты `[0.5, 1, 1.5, 2]`.

Обратите внимание, порт на котором будут доступны метрики и основной порт работы приложения отличаются.

Для проверки запустите `yarn test`.

## Подсказки

[prom-client](https://github.com/siimon/prom-client).

Не забудьте зарегистрировать Histogram в Registry.

У prom-client есть отдельный метод который отдает метрики в JSON.

## Дополнительно

Поднимите `prometheus` для сбора метрик и `grafana` для отображения.

Для запуска используйте docker-compose.
